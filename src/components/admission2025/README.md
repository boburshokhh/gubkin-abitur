# Компоненты приемной кампании 2025/2026

Этот каталог содержит все компоненты, связанные с приемной кампанией 2025/2026 учебного года для Филиала РГУ нефти и газа (НИУ) имени И.М. Губкина в городе Ташкенте.

## Структура компонентов

### 1. AdmissionInfo2025.vue
**Назначение:** Общая информация о приеме
- Название университета и филиала
- Учебный год (2025/2026)
- Форма обучения (очная, дневная)
- Язык обучения (русский)
- Общее количество мест (330)
- Сроки приема документов (16 июня - 1 июля 2025)
- Формат подачи документов (онлайн/офлайн)
- Ссылки на нормативные документы

### 2. Directions2025.vue
**Назначение:** Направления подготовки и специальности
- **I. Специалитет - Технология геологической разведки** (30 мест)
  - Цифровой геоинжиниринг (ГЦ)
- **II. Бакалавриат - Нефтегазовое дело** (120 мест)
  - Бурение нефтяных и газовых скважин (РБ) - 30 мест
  - Эксплуатация и обслуживание объектов добычи нефти (РН) - 30 мест
  - Эксплуатация и обслуживание объектов добычи газа (РГ) - 30 мест
  - Эксплуатация и обслуживание объектов транспорта (ТП) - 30 мест
- **III. Специалитет - Нефтегазовые техника и технологии** (90 мест)
  - Технология бурения нефтяных и газовых скважин (РТ) - 30 мест
  - Разработка и эксплуатация нефтяных месторождений (РС) - 30 мест
  - Разработка и эксплуатация газовых месторождений (РИ) - 30 мест
- **IV. Бакалавриат - Экономика** (30 мест)
  - Экономика и проекты устойчивого развития энергетики (ЭЭ)
- **V. Бакалавриат - Менеджмент** (60 мест)
  - Управление бизнесом в энергетике (ЭМ) - 30 мест
  - Международный менеджмент (БМ) - 30 мест

### 3. DocumentsList2025.vue
**Назначение:** Необходимые документы для поступления
- Заявление (заполненная форма)
- Документ об образовании (аттестат/диплом)
- Паспорт (ID-карта) с требованиями к формату
- Фотографии (4 штуки, 3×4 см, цветные, белый фон)
- Специальные требования для участников олимпиады
- Ограничения по возрасту (16+ лет)

### 4. ApplicationProcess2025.vue
**Назначение:** Пошаговый процесс подачи документов
- **Шаг 1:** Заполнение заявления на сайте
- **Шаг 2:** Отправка документов по email (11 специализированных адресов)
- **Шаг 3:** Получение SMS-подтверждения с регистрационным номером
- **Шаг 4:** Выбор до 3-х конкурсных групп
- Технические требования к файлам
- Контакты технической поддержки

### 5. ExamSchedule2025.vue
**Назначение:** Расписание вступительных испытаний
- **Математика:** 3 июля 2025 (письменно, 3 часа) - для всех направлений
- **Русский язык:** 8 июля 2025 (письменно, 3 часа) - для всех направлений
- **Английский язык:** 11-12 июля 2025 (устно, 30 минут) - только для Экономики и Менеджмента
- Информация о консультациях
- Требования по направлениям

### 6. OlympiadBenefits2025.vue
**Назначение:** Преимущества для участников Губкинской олимпиады
- **Победители и призеры:** прием без вступительных испытаний (технические специальности)
- **Только победители:** прием без экзаменов (Экономика, Менеджмент)
- **Призеры:** 100 баллов за олимпиаду (Экономика, Менеджмент)
- **Участники:** зачет результатов олимпиады (минимум 40 баллов)
- Необходимые документы для подтверждения
- Сроки подачи документов

### 7. ContactInfo2025.vue
**Назначение:** Полная контактная информация
- **Адрес:** г. Ташкент, Мирзо Улугбекский район, ул. Дурмон йули, 34
- **Call-центр:** (+99871) 200-01-56
- **Ответственное лицо:** Копаненко Кристина Александровна
- **Время работы:** Пн-Пт 09:00-18:00, Сб 09:00-17:00
- **Официальный сайт:** https://gubkin.uz
- **Telegram:** @gubkin_uz
- Техническая поддержка и рекомендации

## Использование компонентов

### В комплексной странице
Все компоненты объединены в `/views/Admission2025Page.vue` с:
- Hero-секцией с ключевыми цифрами
- Навигацией по разделам (sticky)
- Плавной прокруткой между секциями
- Финальным призывом к действию

### Отдельное использование
Каждый компонент можно использовать независимо:

```vue
<template>
  <AdmissionInfo2025 />
</template>

<script setup>
import AdmissionInfo2025 from '@/components/admission2025/AdmissionInfo2025.vue';
</script>
```
