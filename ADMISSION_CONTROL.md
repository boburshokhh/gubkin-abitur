# Управление статусом приемной кампании

## 📋 Описание

Система позволяет управлять статусом приемной кампании через переменные окружения `VITE_ADMISSION_OPEN` и `ADMISSION_OPEN`. Когда прием закрыт, пользователи не могут регистрировать новые аккаунты и подавать новые заявления, но могут входить в систему и просматривать уже поданные заявления.

## 🔧 Настройка

### Переменная окружения

```env
# В файле .env
VITE_ADMISSION_OPEN=false  # Прием закрыт
VITE_ADMISSION_OPEN=true   # Прием открыт
ADMISSION_OPEN=false       # Backend-защита закрыта
ADMISSION_OPEN=true        # Backend-защита открыта
```

### Для разработки

1. Создайте файл `.env` в корне проекта
2. Добавьте переменные `VITE_ADMISSION_OPEN=false` и `ADMISSION_OPEN=false`
3. Перезапустите сервер разработки

### Для продакшена (Netlify)

1. Зайдите в Netlify Dashboard
2. Перейдите в Site Settings → Environment variables
3. Добавьте переменную:
   - **Key**: `VITE_ADMISSION_OPEN`
   - **Value**: `false` (для закрытия) или `true` (для открытия)
4. Выполните новый деплой

## 🚫 Что происходит при закрытии приема

### Страница регистрации (`/register`)
- Показывается компонент `AdmissionClosedMessage`
- Форма подачи заявления скрыта
- Данные формы не загружаются при прямом переходе на страницу
- Пользователи видят информационное сообщение

### Страница входа (`/auth`)
- Вход для существующих пользователей остается доступным
- Регистрация новых аккаунтов скрыта
- Прямой API-запрос на регистрацию возвращает ошибку закрытого приема

### Главная страница (`/`)
- Кнопка "Подать документы" заменена на "Прием документов на 2026 год закрыт"
- Кнопка неактивна и не кликабельна

### Хедер сайта
- Ссылка "Подать документы" заменена на "Прием 2026 закрыт"
- В мобильном меню также показывается "Прием 2026 закрыт"

### Личный кабинет
- Кнопка "Подать новое заявление" заменена на "Прием 2026 закрыт"
- Существующие заявления остаются доступными для просмотра

### Навигация дашборда
- Кнопка "Подать новое заявление" скрыта
- Показывается "Прием 2026 закрыт"

## 🔄 Переключение статуса

### Закрыть прием
```env
VITE_ADMISSION_OPEN=false
ADMISSION_OPEN=false
```

### Открыть прием
```env
VITE_ADMISSION_OPEN=true
ADMISSION_OPEN=true
```

**Важно**: После изменения переменной необходимо перезапустить приложение (в разработке) или выполнить новый деплой (в продакшене).

## 🔧 Техническая реализация

```javascript
// Проверка статуса в компонентах
const isAdmissionOpen = import.meta.env.VITE_ADMISSION_OPEN === 'true';

// Условное отображение
<template>
  <button v-if="isAdmissionOpen">Подать заявление</button>
  <span v-else>Прием 2026 закрыт</span>
</template>
```
